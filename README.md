# Webpack-config
______
## Привет! 
Это конфиг webpack для верстки.  
Стабильно работает на Node v16.15.0 \
Для установки, просто склонируй репозиторий к себе и пропиши npm install  
Точка входа index.js  
Папка билда ./build
Скрипты:  
**npm dev** - запуск лайв сервера. Если что-то не работает в данном режиме - значит и в билде будут ошибки!  
**npm build:prod** - продакшн версия билда  
**npm build:dev** - билд разработчика, без хешей, чтобы "посмотреть" как будет выглядеть конечный вариант
